## 7️⃣ Commit Message Convention

> 커밋태그: 내용 #닉네임

ex. `feat : login 파일 추가 #jay'

| command | mean |
| --- | --- |
| feat | 새로운 기능 추가 |
| fix | 버그 수정 |
| docs | 문서 수정 |
| style | 코드 포맷 변경 |
| test | 테스트 코드 추가 |
| refactoring | 코드 리팩토링 |
| chore | 빌드, 패키지 매니저 수정 |
